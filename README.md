# java-mvc-imc-system

Sistema de Gestión y Cálculo del Índice de Masa Corporal (IMC) para el Centro Médico VidaPlena.

## Descripción
Este proyecto implementa un sistema de gestión de pacientes que permite calcular automáticamente el Índice de Masa Corporal (IMC) y determinar su categoría según los estándares de la OMS.

## Tecnologías y Arquitectura
- **Lenguaje**: Java
- **Arquitectura**: MVC (Modelo-Vista-Controlador)
- **Patrón de Diseño**: DAO (Data Access Object)
- **Interfaz**: Java Swing con diseño personalizado y estilizado

## Estructura del Proyecto
- `src/model`: Lógica de negocio y entidades (Persona, Paciente).
- `src/view`: Interfaz gráfica de usuario (Ventanas, Tablas, Botones).
- `src/controller`: Lógica de control y comunicación entre Modelo y Vista.
- `src/dao`: Gestión de acceso a datos.
- `src/app`: Punto de entrada de la aplicación.
